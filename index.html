Curl-to-PHP

将CURL 命令转换为 PHP 代码

将CURL命令行转为PHP代码。
当前可以转换的参数： -d/--data/--data-binary, -F/--form, -H/--header, -I/--head, -u/--user, compressed, ---url and -X/--request
如果发现BUG，请去please GitHub curl-to-php反馈!

该脚本源自Matt Holt的curl-to-Go.
